# ETLtools

`ETLtools` is an R package that provides tools for ETL processes in R.

## Installation

You can install the master version of ETLtools from
[GitHub](https://github.com/vh-d/ETLtools) with:

``` r
devtools::install_github("vh-d/ETLtools")
```
